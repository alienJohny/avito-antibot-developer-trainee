#### Redis

Для решения задачи я использую in-memory базу данных Redis.
Установить ее можно следующим образом.


```shell
wget http://download.redis.io/releases/redis-5.0.8.tar.gz
tar xzf redis-5.0.8.tar.gz
cd redis-5.0.8
make

src/redis-server
```
